Infrastructure Setup

Requirments:
 - Docker
 - Docker Compose

Deployable Services:
 - Mesos Master
 - Mesos Slave
 - Marathon
 - Zookeeper
 - Docker Registry
 - Jenkins

How to Run:
 - docker-compose up
 
How to access the services:
 - Mesos Master http://localhost:5050
 - Marathon http://localhost:8080
 - Jenkins http://localhost:8081
